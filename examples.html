Skeletize.js

Create skeleton screens. So easy, it's eerie.

This plugin is very simple—more of a helper. It will be extended incrementally, but for now just serves the purpose of initializing pages leveraging Ajax content to have skeleton screens. In addition, this plugin exposes a fluent API for easily building skeleton screens on the fly. Follow the steps below and enjoy!

1. Download

Simply install Skeletize.js via Bower:

bower install https://github.com/picklepilot/Skeletize.js.git

2. Place into code

Include the css file in your <head> tag.

<link rel="stylesheet" href="/bower_components/skeletize.js/skeletize.min.css">

Include the js file before the closing of your <body> tag.

<link rel="stylesheet" href="/bower_components/skeletize.js/skeletize.min.js">

3. Initialize

Since Skeletize.js is more of a helper to call on elements on demand, initialization is just to register the main caller.

let skeletize = new Skeletize();

4. Usage Examples

Basic usage:

skeletize.setTarget('.target-element').create();

Add a message to the skeleton screen via data-skeletize-text attribute:

<div class="target-element" data-skeletize-text="Loading new stuff..."></div>
